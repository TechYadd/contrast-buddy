# Contrast Buddy

A tiny WCAG contrast checker built with React + Vite.

## Features

- Computes contrast ratio using WCAG relative luminance
- AA/AAA pass/fail for normal and large text
- Accessible preview area + copyable CSS snippet

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
